# 신동와인 — 채널×브랜드 수익성 리뷰 템플릿

> 대표님 월간 보고용. "매출이 아니라 **진짜 남는 장사**"를 채널·브랜드 단위로 가른다.
> 한 줄 호출: `@templates/shindong-wine/wine-margin-review.md 관점으로`

---

## 이 템플릿이 강제하는 것

- 숫자만 있는 줄은 **미완성**으로 본다. 모든 핵심 숫자 옆에 "그래서 무슨 의미" 한 줄을 붙인다.
- 매출만 보지 않는다. 반드시 **마스터·채널마진과 교차**하여 도착원가·채널수수료를 뺀 **기여이익**까지 낸다.
- TOP만 나열하지 않는다. **왜 위로 왔는지(공통점)** + **부진·마진누수·과재고 위험군**을 함께 짚는다.
- 회의용 코멘트를 **미리 3종(숫자형·흐름형·헤드라인형)** 뽑아 대표님이 고르게 한다.

## 사용 데이터 (반드시 교차)

| 파일 | 쓰임 |
|------|------|
| `shipments.csv` | 출고매출 (채널·브랜드·SKU·일자·금액) — 매출의 원천 |
| `wine_master.csv` | 도착원가(arrival_cost_krw)·환율·정가·tier — **원가 교차** |
| `channel_margin.csv` | 채널별 입점 수수료/마진율 — **수수료 교차** |
| `promotions.csv` | 행사·신제품 기간 — 매출 변동의 **원인 추정** |
| `inventory.csv` | 재고·빈티지·소진일수 — **재고 위험** 연결 |

**기여이익 계산식 (라인 단위):**
```
기여이익 = 출고매출(amount_krw)
          − 채널마진(amount_krw × channel_margin_rate)
          − 도착원가(arrival_cost_krw × qty)
기여이익률 = 기여이익 ÷ 출고매출
```

## 출력 구조

### 1. 한눈 요약 (KPI 카드 4장)

- 총 출고매출 / 총 기여이익 / 평균 기여이익률 / 전월 대비(데이터 있으면)
- **각 카드 아래 한 줄 해석 강제** — 예: "매출은 늘었지만 이익률은 떨어짐 → 저마진 채널 비중↑ (추정)"

### 2. 채널별 수익성 (표 + 해석)

표 컬럼: 채널 / 출고매출 / 채널마진액 / 도착원가 / **기여이익** / **기여이익률** / 매출순위 / 이익순위

- **매출순위와 이익순위가 어긋나는 채널을 굵게** 표시하고, 어긋난 이유를 3~4줄로:
  - 백화점은 매출은 크지만 수수료(32~34%)가 이익을 깎는다
  - 직거래(HORECA·와인샵)·온라인은 매출은 작아도 수수료가 낮아 이익률이 높다
  - "매출을 백화점에 의존하지만 정작 남는 건 직거래" 구조인지 데이터로 판정

### 3. 브랜드별 수익성 (표 + 해석)

표 컬럼: 브랜드 / 출고매출 / **기여이익** / **기여이익률** / 대표 SKU / tier

- **매출 1위 브랜드 ≠ 이익 1위 브랜드**인지 명시. 어긋나면 그 자체가 헤드라인.
- 이익률 상위 브랜드의 **공통점**(엔트리·중가 회전형인지, 직거래 비중인지)을 1~2줄.
- **이익률 하위 위험군**: 매출은 크나 이익률이 평균 미만인 브랜드 → "볼륨만 큰 저효율" 경고.

### 4. 채널×브랜드 교차 (선택)

- 같은 브랜드도 채널에 따라 이익률이 갈린다. 한 브랜드를 어느 채널에 밀어야 남는지 1~2줄.

### 5. 대표 보고용 코멘트 (3종 중 택1 — 미리 제시)

- **숫자형**: "매출 N억 중 기여이익 N억(이익률 N%). 매출 1위 ○○, 이익 1위 △△."
- **흐름형**: "매출은 백화점이 끌지만 이익은 직거래가 만든다. 채널 믹스 재검토 필요."
- **헤드라인형**: "잘 팔리는 와인과 남는 와인은 다르다 — △△를 더 밀어야 한다."

### 6. 액션 (4열 표)

| 담당 | 액션 | 기한 | 우선순위 | 성공 기준 |
|------|------|------|----------|-----------|
| (예) MD | 저이익률 ○○ 채널 믹스 조정 | 차월 | 상 | 이익률 +Np |

## 검출 규칙 (수치 기준)

- **마진 누수 경보**: 브랜드/채널 기여이익률 < 전체 평균 − 5%p → 위험군 표기
- **저효율 볼륨 경보**: 매출 상위 30% 이면서 이익률 하위 30% → 굵게 + "볼륨만 큰" 표기
- **매출↔이익 순위 역전**: 매출순위와 이익순위 차이 ≥ 2계단 → 헤드라인 후보
