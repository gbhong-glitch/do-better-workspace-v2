## 상황 설정

**수요일 오전 9시 40분. 신문로 사무실, 2층.**

창밖으로 경희궁 돌담이 보인다. 오늘은 내부 전체 주간 경영 리뷰가 있는 날이다. 100호를 막 내보냈고, 상반기를 마무리하는 시점이다. 책상 위에는 어제 각 팀이 따로 올린 파일 세 개가 열려 있다.

슬랙 알림이 울린다.

> **효선님**: 대표님, 5월 손익 마감해서 올렸어요 (`finance_monthly.csv`)
>
> 부문별로 4월·5월 나란히 정리했는데요
> 라이선스가 5월에 확 빠진 게 눈에 걸려요
>
> 리뷰 때 "그래서 5월 어땠어요?" 물으실 거잖아요
> 부문 숫자만 던지면 또 "그래서 뭐가 문제냐"로 돌아오니까... 거기까지 정리되면 좋겠어요

**(속으로)**: 라이선스가 빠진 게 단발이라 그런 건지, 추세인지부터 봐야 한다. 리뷰 자리에서 분명 같은 질문이 나올 텐데.

슬랙이 또 온다.

> **유원님**: 대표님! 유통팀 주간 출고/판매도 올라왔어요 (`distribution_weekly.csv`)
> 편집 쪽 주간 현황(`editorial_weekly.csv`)도 같이 묶어뒀습니다
>
> 6월 첫 주에 %Arabica 호가 갑자기 확 나갔거든요
> 그 주에 저희가 카페 컬래버 SNS 공개한 날이랑 겹치는 것 같아서요
> 콘텐츠가 판매로 이어진 거면 다음 호 기획에 바로 반영하려고요

**(속으로)**: 세 팀이 각자 다른 양식으로 올렸다. 편집은 주차(W)로, 재무는 월로, 유통은 날짜 범위로. 매번 이걸 손으로 맞춰 붙이는 게 일이었지.

- 진짜 마감: 오전 11시 내부 주간 리뷰
- 체감 마감: 10시 반 (들어가기 전에 한 번 훑어야 함)
- 실제 가용 시간: 지금부터 한 시간
- 받은 파일: 편집장 주간 현황, 재무 월간 손익, 유통 주간 출고/판매, 제품 마스터

**Claude Code가 없었다면?**
- 세 팀 파일 열어서 기간 기준 맞추기 (주차↔월↔날짜) (1시간)
- 부문별 손익 피벗 + 전월 비교 (1시간)
- 유통 판매를 제품·채널로 쪼개고 원가·수수료 붙여 마진 계산 (2시간)
- 편집 현황에서 '대표 검토 대기' 추려서 안건화 (30분)
- 한 장으로 정리 (1시간)
- 총 **5시간+, 리뷰는 내일로 미룸**

**Claude Code가 있다면?**
- 세 파일 한 번에 취합 요청 (30분)
- 마진까지 교차하고 추가 질문 (30분)
- 총 **1시간, 11시 리뷰 그대로 진행**

자, 시작해보자.

## 미션 1: 세 팀 자료를 하나의 주간 경영 리뷰로 취합

효선님과 유원님이 따로 올린 세 파일. 리뷰 들어가기 전에 하나로 묶어야 한다.

> **효선님**: 대표님 리뷰 첫 질문이 늘 정해져 있잖아요
> "5월 어땠고, 이번 주 뭐가 움직였냐" 이거요
>
> 근데 저희 자료가 기준이 다 달라요
> 저는 월, 유통은 주, 편집은 주차(W)...
> 그냥 합치면 숫자가 안 맞으니까, 기간 기준부터 정리되면 좋겠어요

**(속으로)**: 매번 여기서 막혔다. 세 양식을 손으로 맞추다 시간 다 보내고, 정작 "그래서 뭐" 판단은 회의 자리에서 즉석으로 했지. 이번엔 취합을 맡겨보자.

**분석 요청**:
```
@finance_monthly.csv @distribution_weekly.csv @editorial_weekly.csv
@templates/magazine-b/weekly-management-review.md 관점으로
이번 주 경영 리뷰 한 장 만들어줘. 세 자료 기간 기준이 다르니 먼저 맞춰서 명시하고,
부문 손익(라이선스 왜 빠졌는지)·주간 판매 급등 주·대표 검토 대기 안건까지 묶어줘.
```

**결과 확인 후 추가 질문 예시**:
```
"6월 첫 주 판매 급등이 편집 쪽 %Arabica SNS 공개일이랑 겹치는지 데이터로 확인해줘. 콘텐츠가 판매를 끌었다는 근거가 되나?"
```

## 미션 2: 채널×제품 마진 — 매출 1등이 마진 1등인가

리뷰 30분 전. 유원님한테 슬랙이 하나 더 온다.

> **유원님**: 대표님, 하나만 더요
> 판매액 표 보다 보니까 궁금한 게 생겼어요
>
> 100호랑 디자인 아이콘 호(Leica·FREITAG)가 매출은 잘 나오는데
> 이게 진짜 남는 장사인지 모르겠어요
> 해외로 많이 나가는 호는 수수료가 크잖아요
>
> 매출 순위랑 실제 남는 순위가 다를 수도 있을 것 같은데... 확인 가능할까요?

**(속으로)**: 좋은 질문이다. 광고 없이 판매로만 버는 구조라, "매출 1등"이 아니라 "마진 1등"이 진짜 봐야 할 숫자다. 채널 수수료까지 붙여봐야 보인다.

**분석 요청**:
```
@distribution_weekly.csv @product_master.csv
@templates/magazine-b/weekly-management-review.md 의 "채널×제품 마진" 관점으로
판매액을 순마진(채널 수수료·제작원가 반영)으로 환산해서,
채널별 마진율이랑 제품별 매출순위 vs 마진순위 비교해줘. 순위 역전되는 호는 강조해서.
```

**결과 확인 후 추가 질문 예시**:
```
"매출은 높은데 마진 순위가 떨어지는 호들, 직판(자사몰) 비중을 올리면 마진이 얼마나 회복되는지 가정해서 보여줘"
```
